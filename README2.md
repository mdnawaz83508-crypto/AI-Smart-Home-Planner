# 🚀 Space Shooter Game

A 2D browser space shooter game where players control a spaceship and destroy alien ships.

## Features

- Spaceship movement
- Alien enemies
- Laser shooting
- Collision detection
- Score system
- Lives system
- Particle explosions
- Animated stars
- Increasing difficulty

## Controls

Keyboard:
- Left Arrow = Move Left
- Right Arrow = Move Right
- Space = Shoot

## How To Run

1. Download the project.
2. Open index.html in your browser.
