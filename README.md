# 🏠 AI Smart Home Planner

An AI-based home design assistant that helps users create smart and modern house design ideas.

## 🚀 Features

- 🏠 AI room design suggestions
- 💰 Budget-based home planning ideas
- 🎨 Different interior design styles
- ✨ AI house design prompt generator
- ▶️ YouTube house design video search
- 🖼️ Google image search for design inspiration
- 📱 Mobile-friendly design
- 🌐 Works directly in a web browser


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
